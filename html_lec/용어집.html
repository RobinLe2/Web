HTML 용어집

▶Doctype
문서가 HTML5임을 선언 (<!DOCTYPE html>).
<!DOCTYPE html>
문서 최상단 선언
▶html, head, body
루트 요소(<html>), 메타데이터(<head>), 콘텐츠(<body>).
<html lang="ko">
  <head>…</head>
  <body>…</body>
</html>

루트, 메타, 콘텐츠 구조

▶Heading
<h1>~<h6> 문서 제목.
<h1>제목</h1>

제목

▶Paragraph
<p> 문단.
<p>문단</p>

문단

▶Division (div)
블록 컨테이너 (<div>).
<div>내용</div>
내용
▶Span
인라인 컨테이너 (<span>).
<span style="color:red;">텍스트</span>
텍스트
▶Link
<a href="url"> 하이퍼링크.
<a href="https://example.com">클릭</a>
클릭
▶Image
<img src="..." alt="..."> 이미지 삽입.
<img src="https://via.placeholder.com/60" alt="예시">
▶List
<ul>/<ol> + <li> 목록.
<ul><li>A</li></ul>
  • A
▶Table
<table>,<tr>,<th>,<td> 표.
<table border="1"><tr><td>X</td></tr></table>
X
▶Form
<form> 입력 그룹.
<form><input></form>
▶Input
다양한 타입 (text, number, file 등).
<input type="text" placeholder="이름">
▶Select & Option
<select> + <option> 드롭다운.
<select><option>A</option></select>
▶Textarea
여러 줄 입력 (<textarea>).
<textarea rows="2" cols="20"></textarea>
▶Checkbox
<input type="checkbox"> 체크박스.
<input type="checkbox">
▶Radio
<input type="radio"> 라디오 버튼.
<input type="radio" name="r">
▶Datalist
<datalist> 자동완성 목록.
<input list="list"><datalist id="list"><option>A</option></datalist>
▶Label
<label> 폼 요소에 설명(레이블) 제공.
<label for="username">아이디</label>
<input type="text" id="username">
▶Button
<button> 클릭 가능한 버튼.
<button type="button" onclick="alert('Hello')">클릭</button>
▶Textarea
<textarea> 여러 줄 텍스트 입력.
<textarea rows="3" cols="30" placeholder="댓글 입력"></textarea>
<input list="list"><datalist id="list"><option>A</option></datalist>

CSS 용어집

▶Position
요소의 배치 방식(static, relative, absolute, fixed).
.box { position: absolute; top: 10px; left: 10px; }
box
▶Flexbox
1차원 레이아웃 모델. 컨테이너와 아이템 정렬·배치.
.container { display: flex; justify-content: space-between; }
1
2
▶Display
요소의 기본 표시 방식(block, inline, inline-block).
span { display: block; }
block span
▶Float
텍스트나 다른 요소가 주변을 감싸도록 좌우로 띄움.
.box { float: left; width:60px; }
box

텍스트가 감싸입니다.

▶Overflow
컨텐츠가 영역을 벗어날 때 처리(hidden, scroll, visible).
.box { width:100px; height:50px; overflow: hidden; }
Overflow 처리 예시 - 긴 텍스트가 잘립니다.
